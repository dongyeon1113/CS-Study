# [강의 요약] 머신러닝 개요 (Overview of ML)

---

### 1. 머신러닝이란? (What is ML?)
* **핵심 정의**: 주어진 데이터나 과거의 경험을 분석하여 프로그램의 성능을 향상시키는 기술.
* **데이터 분석 기법**: 인간이 일일이 규칙을 코딩하는 대신, 데이터 세트를 분석하여 컴퓨터 프로그램을 자동 생성하는 방법론.
* **통계학과의 차이**: 
  - 통계학은 수학적 엄밀함을 강조하는 수학의 한 분야.
  - 머신러닝은 데이터 분석을 위해 **휴리스틱(Heuristic)**이나 **대담한 가정**을 허용하는 공학적 접근.

### 2. AI, ML, DNN의 관계
* **AI (인공지능)**: 지능적으로 행동하는 프로그램을 구축하는 가장 넓은 개념.
* **ML (머신러닝)**: 지능의 원천을 인간(전문가)이 아닌 **데이터**에서 얻는 AI의 하위 분야.
* **DNN (심층신경망)**: 머신러닝의 한 종류로, 신경망 구조를 통해 데이터 중심의 지능을 구현 (딥러닝).



---

### 3. 머신러닝 문제의 주요 유형

####  지도 학습 (Supervised Learning)
> 정답(**Label**)이 포함된 데이터셋을 학습하여 미래를 예측.
* **분류 (Classification)**: 데이터를 정해진 범주로 나누는 문제 (예: 스팸 메일 분류, 얼굴 인식).
* **회귀 (Regression)**: 변수 간의 관계를 파악하여 연속적인 수치를 예측 (예: 주식 예측, 마케팅 분석).




####  비지도 학습 (Unsupervised Learning)
> 정답(**Label**)이 없는 데이터를 사용하여 숨겨진 구조나 패턴을 발견.
* **군집화 (Clustering)**: 유사한 특징을 가진 데이터끼리 그룹화 (예: 고객 세그먼트 분석, 검색 결과 그룹핑).
* **차원 축소 (Dimension Reduction)**: 정보 손실을 최소화하며 데이터의 복잡성을 줄임.



---

### 4. 머신러닝을 위한 필수 기반 지식
성공적인 머신러닝 학습을 위해 필요한 학문적 기초:
1. **수학 (Mathematics)**: 확률과 통계(베이즈 정리 등), 선형대수(행렬 계산), 최적화(경사하강법).
2. **프로그래밍 (Programming)**: 동적 계획법(DP), 분할 정복, 알고리즘 시간 복잡도 분석.
3. **도메인 지식 (Domain Knowledge)**: 데이터가 발생하는 분야(금융, 의료, 게임 등)에 대한 이해.
