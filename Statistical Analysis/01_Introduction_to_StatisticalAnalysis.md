# 📊 통계 분석 1주차: 통계 분석 소개 (Introduction to Statistical Analysis)

## 1. 통계학이란 무엇인가?
불확실성과 변동성이 존재하는 현실 세계에서 **데이터의 의미를 파악하는 과학적 학문** 입니다.
데이터를 수집, 정리, 분석, 해석하여 단순한 숫자를 **유의미한 정보와 통찰(Insight)** 로 변환하고, 객관적인 **데이터 기반 의사결정**을 지원합니다.

* **주요 역할**: 데이터 요약, 데이터 시각화, 패턴 찾기

## 2. 통계학의 두 가지 핵심 분류
* **기술 통계 (Descriptive Statistics)**: 수집된 데이터의 특징을 요약하고 묘사하는 통계 기법. (예: 평균, 중앙값, 표, 그래프)
* **추론 통계 (Inferential Statistics)**: 표본(Sample)의 데이터를 분석하여 모집단(Population)의 특성을 추측하고 결론을 도출하는 통계 기법. (예: 가설 검정, 신뢰구간 추정)

## 3. 데이터 분석 4단계 프로세스
1. **문제 정의 (Define the Problem)**: 연구 목표를 설정하고, 무엇을 알고 싶은지 명확히 합니다.
2. **데이터 수집 (Collect Data)**: 문제를 해결하기 위해 적절한 샘플링 계획을 세우고 신뢰할 수 있는 데이터를 확보합니다.
3. **데이터 요약 (Summarize Data)**: 수집한 데이터를 표, 그래프 등으로 시각화하고 구조화합니다. (기술 통계 활용)
4. **분석 및 결론 (Analyze & Conclude)**: 분석 모델을 적용하여 통계적 결론을 도출하고 실무적 의미를 해석합니다. (추론 통계 활용)

## 4. 통계학 주요 용어 ⭐️ 
| 용어 | 영문 | 의미 | 기호 예시 |
| :--- | :--- | :--- | :--- |
| **모집단** | Population | 분석의 대상이 되는 전체 집단 | |
| **표본** | Sample | 모집단에서 실제 조사를 위해 추출된 일부 집단 | |
| **모수** | Parameter | **모집단**의 특성을 나타내는 고정된 값 (알려지지 않음) | 모평균($\mu$), 모표준편차($\sigma$) |
| **통계량** | Statistic | **표본**에서 계산된 값 (추출할 때마다 변할 수 있음) | 표본평균($\bar{x}$), 표본표준편차($s$) |

## 5. 통계 분석 시 주의할 점 (인지 편향)
데이터를 해석할 때 직관에 의존하거나 잘못된 인과관계를 설정하면 오류가 발생합니다.
* **생존자 편향 (Survivorship Bias)**: 성공한 사례(눈에 보이는 데이터)만 보고, 실패한 사례(사라진 데이터)를 간과할 때 발생하는 오류. (예: "빌 게이츠를 보니 대학은 필요 없다.")
* **상관관계와 인과관계의 혼동 (Correlation vs. Causation)**: 두 현상이 함께 변한다고 해서, 하나가 다른 하나의 '원인'이라고 단정 짓는 오류. (예: 초콜릿 소비량이 많은 나라가 노벨상 수상자가 많다는 통계)
